# Aggregate tables

Anonymized, aggregate-only tables backing the research note. No row describes an
identifiable individual; participant references use stable pseudonyms only.

| File | Contents |
|---|---|
| `engagement_rates.csv` | Engagement and voluntary re-enrollment rates (numerator / denominator / %), by cohort. Backs Evidence 1. |
| `disclosure_counts.csv` | Counts of structured answers, extracted knowledge nodes, and voluntary disclosures by type. Backs Evidence 2 and 4. |
| `expressed_vs_captured.csv` | Preference-bearing disclosures expressed vs. structured by the capture pipeline, by channel. Backs Evidence 4. |

Figures are transcribed from the cohort analyses of the two Kay.Eco cohorts. Underlying
transcripts contain PII and are not published (see the Data availability statement in the
top-level README).
